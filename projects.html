Yahtzee

Yahtzee game using java and java swing Description: The project has many different classes that are used to represent objects in the game. You are able to run the game by putting it in your ide that is compatible with Java and Java Swing. Run it select how many players or what kind of computer you would like to play against and play the game. You play by creating sets of dice by rolling, saving and unsaving dice. The game iterates through every player and ends after every set is made.

Tucson Az Crime Analysis

Data analysis of crime in Tucson Az using libraries such as Pandas,Matplotlib,Seaborn, SciPy ,scikit-learn, and NumPy. Uses public city data to understand crime in the city and what recordable factors have an impact on the characteristics of crime. Using connections found in the initial data exploration the report idetntifies possible methods to prevent certain types of crime in the city.

Alphabet Soup

Web application built from HTML,CSS, and Javascript that creates encrypted text. Alphabet Soup uses both a classic square cipher and caesar cipher. User controls the text to be encrypted and the Number fo shifts used in the square cipher.

AzHoldEm

A 2-10 player poker game. This rendition excludes betting but tracks the player entry fees, the running pot, and each players current winnings. Winners are determined by the highest hand from the calculated possible hand types.
